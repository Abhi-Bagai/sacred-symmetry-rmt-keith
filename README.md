# Assets folder

Place static images here, for example:

- **Hero image** – reference in `index.html` if you replace the hero placeholder with an `<img>` (e.g. `assets/hero.jpg`).
- **Therapist headshot** – for the About section (e.g. `assets/headshot.jpg`).
- **og-image.jpg** – for Open Graph/social sharing (recommended size ~1200×630 px).

The site works without any images; the current layout uses placeholder divs.
